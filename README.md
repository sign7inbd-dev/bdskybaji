 ├─ games.html                 → সব গেমের লিস্ট (স্লট, লাইভ ক্যাসিনো, ক্র্যাশ ইত্যাদি)
 ├─ sports.html                → স্পোর্টস বেটিং পেজ (নতুন যোগ করা উচিত)
 ├─ live-casino.html           → লাইভ ক্যাসিনো পেজ
 ├─ profile.html               → প্রোফাইল + KYC + পেমেন্ট
 ├─ deposit.html               → ডিপোজিট পেজ (bKash, Nagad, Rocket...)
 ├─ withdraw.html              → উইথড্র পেজ
 ├─ promotions.html            → বোনাস + অফার পেজ
 ├─ css/
 │   ├─ style.css              → মেইন স্টাইল (ডার্ক থিম, রেসপনসিভ)
 │   ├─ responsive.css         → মোবাইল + ট্যাবলেটের জন্য
 │   └─ animate.css            → অ্যানিমেশন (অপশনাল)
 ├─ js/
 │   ├─ main.js                → সাধারণ জাভাস্ক্রিপ্ট (মেনু, মোডাল...)
 │   ├─ auth.js                → লগইন/রেজিস্টার লজিক
 │   ├─ games.js               → গেম ফিল্টার + লোডিং
 │   └─ api.js                 → API কলের ফাংশন (ফেক/রিয়েল)
